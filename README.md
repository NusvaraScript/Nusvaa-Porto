⚙ Installation

```
git clone https://github.com/NusvaraScript/Nusvaa-Porto.git
cd Nusvaa-Porto

composer install
npm install

cp .env.example .env
php artisan key:generate
```

Configure your database in ".env", then:

```
php artisan migrate
npm run dev
php artisan serve
```
